Dataset Summary

The dataset contains 243 records of forest fires in Algeria, with the following features:

  • Date Columns: day, month, year
  • Weather Conditions: Temperature, Relative Humidity (RH), Wind Speed (Ws), Rain
  • Fire Weather Index Components: Fine Fuel Moisture Code (FFMC), Duff Moisture Code (DMC), Drought Code (DC), Initial Spread Index (ISI), Build-Up Index (BUI), Fire Weather Index (FWI)
  • Classification: Classes (fire or not fire)
  • Region: Geographic region (0 or 1)

Exploratory Data Analysis (EDA) Summary

The EDA was conducted to understand the distribution and relationships of the features in the dataset. Key steps included:

  • Data Cleaning: Ensuring no missing values and consistent data types.
  • Visualizations: Creating plots such as histograms, box plots, scatter plots, and correlation heatmaps to identify patterns and relationships among features.
  • Feature Engineering: Transforming raw data into meaningful features for machine learning models.

Model Building Summary

The model building process involved:

  • Data Splitting: Dividing the data into training and testing sets.
  • Model Selection: Choosing appropriate machine learning algorithms (e.g., logistic regression, decision trees, random forests, etc.).
  • Model Training: Fitting the model to the training data.
  • Model Evaluation: Assessing model performance using metrics such as mean absolute error, r2_score.
